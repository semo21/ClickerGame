# Economy System Spec (Design)

## 목표/의도

- 짧은 세션에서도 성취감, 오프라인 보상으로 복귀 유도.

## 핵심 루프

1. 클릭 -> Currency 획득
2. 업그레이드 구매 -> CPS 증가
3. 오프라인 보상 수령 -> 루프 재진입

## 수식/규칙

- CPC = f(UpgradeLevel) = Base \* L
- CPS = f(UpgradeLevel) = Base \* L / 2
- 오프라인 보상 = CPS \* 오프라인시간(초) / 30
  - 제한: 8시간

## 컴텐츠 맵(요약)

- 업그레이드 10티어, 각 5레벨 / 가격 곡선, 잠금 조건

## UX 플로우

- 첫 진입 -> 튜토리얼 -> 첫 보상 토스트 -> 업그레이드 안내
