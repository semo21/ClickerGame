# 게임 기획 문서

1. 목표

- 클릭을 하면 재화가 오르는 Clicker Game 개발 목표

## 1-1. 주요 목표

- 클리커 게임의 Steam, Android 출시.
- 현재는 포트폴리오 목적.

  - 추후 추가기능을 추가하며 상용 게임으로 수정할 용의 있음.
  - 단, 서버를 사용할 생각은 아직 없음.

## 1-2. 세부 목표

- 코드의 구조 정립.
  - 정확히 표현하자면, 디자이너가 리소스를 교체하여 세부조정만으로도 온전한 기능을 하는 코드구조를 구성하는 것이 목표.
  - 즉, 게임의 코드 구조를 타인이 사용할 수 있는 하나의 '패키지 모듈'로서 구현하려 함.
- UE의 개발 구조 파악
- Steam의 출시과정 파악

# 2. 개발에 사용한 소스

- 프로그래밍은 GPT, UE forum을 참고.
- 디자인 소스는 Fab에서 다운받아 사용.

## 2-1. 개발 참고 소스

- 주로 GPT를 이용하여 디자인패턴과 합리적인 구조를 지속적으로 리팩터링함.
- GPT의 반응 신뢰도가 떨어진다고 느낄 땐 UE Forum 또는 reddit같은 커뮤니티 참고.

## 2-2. 디자인 소스

- Fab에서 구매해둔 에셋 중 골라 사용.
